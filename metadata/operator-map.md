# Operator Map

Future result files should use the computer name in the first column. The second column is the LaTeX label used in plots and paper-facing tables.

In computer names, `p` and `m` denote the `C_lq^+` and `C_lq^-` combinations, and `ll` denotes the lepton-flavour-universal lepton indices.
